CSS POSITIONING:
--------------------
Positioning controls where an element appears on the webpage.
it allows us to move elements from their normal; location.

Syntax:
------
postion:value;

Types of position:
-------------
1. static
2. relative
3. absolute
4. fixed
5. sticky

Position 1: Static:
-------------------
Static is the default position of every HTML element.
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        .box{
            width: 100px;
            height: 100px;
            border:2px solid black;
            position: static;
        }
    
    </style>
</head>
<body>
    <div class="box">Box 1</div>
    <div class="box">Box 2</div>
    <div class="box">Box 3</div>
</body>
</html>

everything appears normally.
nothing changes.

Important point:
----------------
for static elements:
top
left
right
bottom
do not work.

Position 2 - Relative:
-------------------
moves an element relative to its original position.

------------------------------------------------
The CSS position property specifies how an html should be positioned on a webpage.

The position property tells the browser:
where should this element be placed, and what should it use as its reference.

Syntax:
------------
selector{
position:value;
}
 eg:
-------
.box {
position:relative;
}

---------------------

----------------------------

Position: Relative
----------------
An element with position:Relative is positioned relative to its original location.
The element first occupies its normal position.
Then we can move it from that original position using offset properties.

Syntax:
---------
.box{
position:relative;
top:20px;
right:30px;
}    

 <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        body{
            font-family: Arial, Helvetica, sans-serif;
            background-color: #f4f6f8;
            padding:30px;
        }
        h1{
            text-align: center;
            margin-bottom: 30px;
        }
        .container{
            width:500px;
            margin:40px auto;
            padding:25px;
            border:3px solid black;
            background-color: white;
        }
         .box{
            width:250px;
            padding:25px;
            margin: 15px auto;
            border:2px solid black;
            text-align: center;
            font-size:20px;
            font-weight:bold;
        } 
        .box1{
            background-color: lightcoral;
        }
        .box2{
            background-color: lightblue;
            position: relative;
            top:40px;
            left:120px;
        }
        .box3{
            background-color: lightgreen;
        }
    </style>
</head>
<body>
    <h1>Normal Document</h1>
    <div class="container">
    <div class="box box1">Box One</div>
    <div class="box box2">Box Two</div>
    <div class="box box3">Box Three</div>
    </div>
</body>
</html>

Characteristics:
-------------------
A relative positioned element:
->Remains in the normal document flow
->keeps its original space reserved
->can be moved from its original space
->does not cause nearby elements to occupy its old space
->can become the reference parent for absolute child.

Tech Example:
----------------
Button Lift Effect Using Relative:
-------------------------------------
Many websites make a button move slightly upward when the user hovers over it.
This gives the feeling that the button is lifting.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        body{
            font-family: Arial, Helvetica, sans-serif;
            background-color: #111827;
            min-height: 100vh;
            display: flex;
            justify-content: center;
            align-items: center;
        }
        .buy-button{
            padding: 16px 35px;
            border:none;
            border-radius: 8px;
            background-color: #22c55e;
            color: white;
            font-size: 18px;
            font-weight: bold;
            cursor: pointer;
            position: relative;
            transition: 0.3s;
        }
        .buy-button:hover{
            top:-6px;
        }
    </style>
</head>
<body>
    <button class="buy-button">Buy Now</button>
</body>
</html>
